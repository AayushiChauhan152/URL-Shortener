# URL-Shortener

<h3>Dependencies</h3>
<ul>
  <li>nodemon</li>
  <li>ejs</li>
  <li>dotenv</li>
  <li>express</li>
  <li>mongoose</li>
  <li>nanoid</li>
</ul>

## Install Dependencies

```bash
npm install 
```

## Run

```bash
nodemon urlShortener.js
```
